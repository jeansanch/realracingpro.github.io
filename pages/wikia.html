Links da Wikia

Esta página reune os principais links da wikia explicando o que é possível encontrar na página informada.

  • Upgrades Strategy for 100% informa a melhor estratégia para se completar todas as séries gastando o menos possível. Nele você encontra:

    - Qual carro comprar

    - Quanto se gasta e em que gastar

    - se o investimento é rentável

    - se o investimento não é rentável

    - e muito mais

  • Para informações sobre carros, você tem 3 opções:
    • Cars trás uma lista com todos os carros do jogo, separando por montadora, preço de compra, etc.
    • Cars (Fully_Upgraded_with_R$) trás a mesma lista mas com o resultado de um upgrade total usando apenas R$, informando:

      - CD Máximo

      - Número de Upgrades

      - informações tecnicas (top speed, grip, etc)

      - CUSTO de upgrade

    • Cars (Fully_Upgraded) mostra a mesma informação da anterior, mas usando R$ e OURO para deixar o carro full.
